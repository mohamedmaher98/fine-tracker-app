# Personal Finance Manager

## Problem
A web application to track daily income and expenses,
providing monthly summaries to help users manage their finances better.

## MVP Features
- [ ] User registration & login
- [ ] Add income / expense
- [ ] Transactions list (monthly)
- [ ] Monthly summary dashboard

## Tech Stack
- Frontend: Vue 3
- Backend: Spring Boot (Java)
- Database: SQL Server
